Farmerce

Project information

  • Category: Web Application
  • Course: ENTERPRISE PROGRAMMING (ENTPROG)
  • Project date: 2021
  • Project URL: https://www.youtube.com/watch?v=YDz4Xjxp4Rw

What does Farmerce do? It provides an e-commerce website that will help support our local farmers that will solve their problems during the pandemic. Netizens would be able to post and purchase products on our website for easy transaction of the product of their choice. The farmers will be able to use our website to view, create and update the products that they have listed. Only the developers or administrators would be able to delete an existing product.
