PreSchool Portal

Project information

  • Category: Web Application
  • Course: Rapid Application Development (RAPIDEV)
  • Project date: 2023
  • Project URL: https://www.youtube.com/watch?v=C7isQMqnpiE

Made within the low-code platform environment of OutSystems, this web application is crafted to facilitate quick and seamless pre-school appointments. The user-friendly interface and streamlined processes contribute to a more efficient scheduling experience for both parents and educators. Parents can easily book appointments for their children, while educators can manage and organize the schedule with minimal effort.
