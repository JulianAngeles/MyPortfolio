MinMax Utilization and Process Utilization

Project information

  • Category: Dashboard using PowerBI
  • Course: Practicum (at Asurion)
  • Project date: 2023

I used PowerShell and PowerBI to retrieve utilization data from the hosts monitored on DynaTrace by our team. This involved calling the DynaTrace API in PowerShell and exporting the data to an Excel file. This Excel file served as the data source for the PowerBI dashboard we created. The dashboard played a key role in presenting essential information to team members, pinpointing servers that needed attention and action.
